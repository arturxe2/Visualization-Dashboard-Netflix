Data used for the realization of this visualization is available in Kaggle and contains information about movies and TV shows offered in Netflix. Moreover, other information such as movies puntuation made by users and posters of the items has been obtained from The Movie Database. The preprocessing applied to the data and other information about the decisions that have been taken in order to do the dashboard can be found in the document Dashboard_NikolinaEnricArtur.pdf in github. You can also find there the final dataset used, the dashboard in .twb format and the code used to modify the data.
